# AI-Restaurant-Chatbot
AI-powered restaurant chatbot built using Python

## Description
A Python-based conversational chatbot for restaurant applications that allows users to view menu items, check pricing, and place orders through an interactive command-line interface.

## Features
- View menu and pricing
- Place food orders
- View current order and total
- Handles invalid user inputs gracefully

## Technologies Used
- Python
- Basic conversational logic
- Command-line interface

## How to Run
1. Clone the repository
2. Run:
   python restaurant_chatbot.py
